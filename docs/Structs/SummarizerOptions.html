SummarizerOptions

public struct SummarizerOptions

SummarizerOptions provides a way to configure options specifically for summarization tasks. This struct encapsulates parameters that influence how the summary is generated.

  • temperature

    The sampling temperature to control randomness (values between 0.0 to 1.0).
