LLMDomainRouterProtocol

public protocol LLMDomainRouterProtocol

Protocol defining the behavior of a domain router in the LLMManager system.

  • name

    The name of the domain router, used for logging and identification purposes.

    Declaration

    Swift

    var name: String { get }
  • service

    The associated LLM service used to determine the domain. The service is responsible for processing the request and identifying the domain.

    Declaration

    Swift

    var service: LLMServiceProtocol { get }
  • supportedDomains

    A list of domains that the router supports.

    Declaration

    Swift

    Declaration
