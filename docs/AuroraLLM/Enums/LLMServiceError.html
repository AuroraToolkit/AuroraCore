LLMServiceError

public enum LLMServiceError : Error, Equatable

LLMServiceError defines a set of error types that can occur while interacting with LLM services. These errors provide more granular control and understanding of the failure points within the service.

  • missingAPIKey

    Error thrown when the API key is missing for a service that requires authentication.

    Declaration

    Swift

    case missingAPIKey
  • invalidResponse(statusCode:)

    Error thrown when the response from the API is invalid, typically due to an unexpected status code.

    Declaration

    Swift

  • decodingError

    Error thrown when the service encounters an issue decoding the response data.

    Declaration

  • invalidURL
