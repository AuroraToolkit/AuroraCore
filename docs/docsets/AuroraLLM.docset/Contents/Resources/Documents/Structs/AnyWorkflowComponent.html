AnyWorkflowComponent

public struct AnyWorkflowComponent : WorkflowComponent

A type-erased wrapper for any WorkflowComponent.

AnyWorkflowComponent allows developers to include heterogeneous workflow components in a unified structure. It abstracts the underlying type of the component while still conforming to WorkflowComponent.
