WorkflowComponent

public protocol WorkflowComponent

A protocol representing any component of a workflow.

Components conforming to WorkflowComponent can be tasks, task groups, or any other custom workflow elements. This protocol provides a unified interface to convert components into the Workflow.Component enum.
