ContextManager

public class ContextManager

ContextManager is responsible for managing multiple ContextController instances. It allows adding, removing, switching between contexts, and saving/restoring contexts from disk using tasks.

  • addNewContext(_:llmService:summarizer:)

    @discardableResult
public func addNewContext(_ context: Context? = nil, llmService: LLMServiceProtocol, summarizer: SummarizerProtocol? = nil) -> UUID

    Parameters

    context

    An optional Context object. If none is provided, a new one will be created automatically.
    llmService

    The LLM service used for summarization in this context.
    summarizer

    An optional Summarizer instance to handle text summarization. If none is provided, a default summarizer will be created.

    Return Value

    The unique identifier (UUID) for the newly created ContextController.
